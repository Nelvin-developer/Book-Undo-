# BookUndo

Sell, Borrow, Donate, and Discover Used Books Near You.

## Features

- Google Authentication
- Add Books
- Browse Books
- Search & Filter Books
- Favorites
- My Books
- User Profiles

## Tech Stack

- Next.js
- TypeScript
- Firebase Authentication
- Firestore
- Firebase Storage
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
